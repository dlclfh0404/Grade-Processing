## Grade-Processing
자바를 활용한 성적 처리 프로그램

### 프로그램 알고리즘
1. People 객체를 만들고 성명, 국어, 영어, 수학, 합계, 평균, 석차 변수를 만든다.
2. 각 변수의 getter, setter를 만들어준다.
3. 그 후 printScore() 함수로 각 변수들을 출력해준다.
4. 입력을 받는다.
5. 객체배열을 선언 및 초기화를 하고 성명, 국어, 영어, 수학을 입력 받는다.
6. 그 후 석차 처리를 한 후 출력을 한다.
